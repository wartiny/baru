Dark mode versi web atau desktop

WA web dark mode jadi fitur yang paling ditunggu penggunanya. Berikut cara menggunakan WhatsApp dark mode untuk desktop:

  1. Masuk ke situs WhatsApp web
  2. Klik ikon tiga titik di ujung kanan atas dan pilih settings
  3. Selanjutnya klik theme, pilih dark, dan oke.
