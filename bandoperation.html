Band operation
  1. In power on state, short touch the function key to light up the screen to default page with date and time information;
  2. In the default page, short touch the function key can switch band to different pages. The screen will be turned off if no operation in 5 seconds;
  3. When switching to heart rate and blood pressure interface, the band will automatically start the measurement. The result will be displayed in 40s, and then the screen will be turned off automatically.
