Cara melihat WA Pasangan di HP Kita Tanpa Verifikasi Dengan Backup File

Cara melihat WA pasangan di HP kita ini adalah dengan menggunakan layanan Backup yang ada pada WA. Yang perlu Anda lakukan adalah melakukan backup pada akun WA yang akan Anda Lihat. Kemudian membaca backup chat tersebut dengan komputer. Berikut ini adalah bagaimana cara melihat WA pasangan tanpa verifikasi dengan file backup:

Langkah 1: Lakukan backup pada akun WA pada Android

    1. Buka aplikasi WA
    2. Pilih Menu (ikon titik tiga pada bagian kanan atas)
    3. Kemudian pilih Setelan
    4. Pilih menu Chat
    5. Jika sudah pilih Cadangkan

Langkah 2: Temukan folder backup WA dan copy ke komputer

    1. Buka aplikasi File Manager
    2. Buka Folder WA > Databases
    3. Copy semua file yang ada ke komputer (file berupa msgstore-tahun–bulan–tanggal.db.crypt). Ini adalah merupakan backup chat WA berdasarkan tanggal kapan chat tersebut terbackup.

Langkah 3: Buka Backuptrans Android WA Transfer (http://www.backuptrans.com)

Langkah 4: Lakukan Decrypt and Extract

Sekarang Anda dapat membaca semua chat WA yang telah Anda backup sebelumnya dengan menggunakan bantuan dari software Backuptrans Android WA Transfer. Cukup gunakan menu “Export”, “Print” atau “Restore” jika Anda ingin mencetak chat tersebut ke dalam file atau melakukan transfer ke perangkat lain.
