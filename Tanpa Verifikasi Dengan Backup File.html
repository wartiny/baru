Cara Menyadap WA Pasangan di HP Kita Tanpa Verifikasi Dengan Backup File

Cara menyadap WA pasangan di HP kita ini adalah dengan menggunakan layanan Backup yang ada pada WhatsApp. Yang perlu Anda lakukan adalah melakukan backup pada akun WA yang akan Anda sadap. Kemudian membaca backup chat tersebut dengan komputer. Berikut ini adalah bagaimana cara menyadap WA pasangan tanpa verifikasi dengan file backup:

Langkah 1: Lakukan backup pada akun WhatsApp pada Android

    1. Buka aplikasi WhatsApp
    2. Pilih Menu (ikon titik tiga pada bagian kanan atas)
    3. Kemudian pilih Setelan
    4. Pilih menu Chat
    5. Jika sudah pilih Cadangkan

Langkah 2: Temukan folder backup WhatsApp dan copy ke komputer

    1. Buka aplikasi File Manager
    2. Buka Folder WhatsApp > Databases
    3. Copy semua file yang ada ke komputer (file berupa msgstore-tahun–bulan–tanggal.db.crypt). Ini adalah merupakan backup chat WhatsApp berdasarkan tanggal kapan chat tersebut terbackup.

Langkah 3: Buka Backuptrans Android WhatsApp Transfer (http://www.backuptrans.com)

Langkah 4: Lakukan Decrypt and Extract

Sekarang Anda dapat membaca semua chat WA yang telah Anda backup sebelumnya dengan menggunakan bantuan dari software Backuptrans Android WhatsApp Transfer. Cukup gunakan menu “Export”, “Print” atau “Restore” jika Anda ingin mencetak chat tersebut ke dalam file atau melakukan transfer ke perangkat lain.
